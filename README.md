dotfiles
========

Setting up
----------

Clone the repository to somewhere like ~/.dotfiles and run the setup script:
```
$ git clone git://github.com/ElliottH/dotfiles ~/.dotfiles
$ cd ~/.dotfiles
$ ./setup.rb
```

The script will prompt you, rather than overwriting any existing files.
